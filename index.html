The CSS Box Model

with added cat

It is a truth universally acknowledged that the internet was created by, and is run by, cats. It is also well-known that cats like to sit in boxes. Taken together, these facts have resulted in the creation of the CSS Box Model

The Block Element box

Imagine that HTML block elements are boxes and the content inside an element is a cat. The box takes up the full width of the screen. By default the cat sits next to the left edge of the box. Let's add a h2 header and see how it looks-

<h2>Cat</h2>

Cat

Doesn't look very pretty, does it? Even if I align the text to the centre there's no space top and bottom.

Cat

Add Padding

We can move the cat away from the edges of the box by adding some padding in CSS. We can add padding to all four sides of the box or some of them. The padding is added in the order top-right-bottom-left. If all four sides are the same you only need to pass in one value. if top and bottom are the same and left and right are the same, two values are needed.

.classname{padding: 1 rem;}

Add in text-align:center and it looks much better.

.classname{padding: 1 rem;
text-align:center;}

Cat

Units

You can use absolute or relative values when defining padding.

Absolute Values

Absolute values are not relative to anything else and generally stay the same size regardless of the viewport size.

While you can use any of these values in your code, most of these units are more useful when used for print, rather than screen output. Normally the only absolute value you wil use is px (pixels).

  • centimetres (cm)
  • millimetres (mm)
  • inches (in)
  • pixels (px)

Relative values

Relative units are relative to something else, for example, the parent element's font, or the size of the viewport. Relative units scale relative to everything else on the page.

  • em (font size of the parent)
  • rem (font size of the root element)
  • lh (line height of the element)
  • vw ( 1% of the viewport's width)
  • vh ( 1% of the viewport's height)
Changing the size of the box

Change the size of the box by giving it a width in CSS.

.classname{padding: 1 rem;
text-align:center; width:50%;}

Cat

Add margins

The box is still positioned at the side of the screen. We can reposition it by adding a margin.

.classname{padding: 1 rem;
text-align:center; width:50%; margin:1rem 2rem 3rem 4rem;}

Cat

Grey area is the margin

Margins add space outside the box. Just like padding, you can add a margin to all sides of the box, one side, or some of them. The order is top-right-bottom-left, just like padding.

If you want the box centred, give it a right and left value of auto.

.classname{padding: 1 rem;
text-align:center; width:50%; margin:0 auto:}
